\doxysubsection{Detailed Description}
This file contains documentation and a test program for the custom C queues used in the ME405 library. 

These queues are faster than regular Python based queues and don\textquotesingle{}t allocate memory.

The code in this file is {\bfseries{not}} the source code which makes the C queues work. That code is written in C as the file {\ttfamily cqueues.\+c} and compiled into the Micro\+Python image used in the ME405 course.
