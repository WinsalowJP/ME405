\doxysubsection{Detailed Description}
This file serves as the main entry point for the robotics control system. 

It initializes hardware components including motors, encoders, sensors, and the IMU, loads calibration data, and sets up a cooperative task scheduler. The tasks include PID control (Track\+\_\+1), user control, encoder updates, and button handling.
